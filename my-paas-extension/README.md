# My PaaS - Docker Visual Designer

Drag & drop Docker service designer with auto docker-compose generation for VS Code.

## Features

- Visual canvas for designing Docker services
- Auto-detect application type and generate Dockerfile
- Generate docker-compose.yml from visual layout
- Built-in service templates
- Docker Compose up/down from VS Code

## Usage

1. Open the **My PaaS** sidebar from the Activity Bar
2. Use **My PaaS: Open Canvas** to start designing
3. Add services from templates or auto-detect your app
4. Generate and manage docker-compose.yml
